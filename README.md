# Customer Churn Analysis Dashboard (Tableau + Python)

## Overview
This project analyzes customer churn behavior using Python for data cleaning and Tableau for interactive visualization.

## Tools Used
- Python (Pandas)
- Tableau Public

## Key Insights
- Inactive customers are significantly more likely to churn
- Customers with only one product have higher churn rates
- Churn is highest during early tenure
- High-value customers in certain regions show elevated churn risk

## Business Recommendations
- Improve onboarding experience for new customers
- Increase engagement through targeted campaigns
- Promote cross-selling strategies
- Investigate high-value churn segments
